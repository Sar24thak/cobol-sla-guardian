# Static Code Parser for COBOL

This VS Code extension analyzes COBOL programs using static code analysis.

## Features
- COBOL syntax validation
- Dead code detection
- Feature extraction
- Dashboard visualization

## Commands
- SLA Guardian: Show Dashboard
- SLA Guardian: Extract Features
- SLA Guardian: Clear Diagnostics

## Usage
1. Open a COBOL file (.cbl or .cob)
2. Save the file
3. The analyzer will detect syntax errors and dead code automatically
4. Open the dashboard using the command palette
